# Avenue-StartPage

定位： 起始页，项目展示，搜索聚合，常用地址

# 使用

安装`pnpm`
```
npm install pnpm -g
```

克隆
```
git clone --depth=1 https://github.com/taoran1401/Avenue-StartPage.git
```

安装
```
# 进入项目
cd vue3-skeleton

# 复制配置文件
cp .env.example .env.development

# 安装
pnpm install
```

启动
```
pnpm dev
```

打包
```
pnpm build
```
